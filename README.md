# devops-netology

Репозиторий для выполнения домашних заданий по курсу DevOps.

## Игнорируемые файлы

Корневой файл `.gitignore` исключает из репозитория:

- `.idea/` — каталог с точным названием `.idea`;
- `.vscode/` — каталог с точным названием `.vscode`;
- `.DS_Store` — файлы с точным названием `.DS_Store`;
- `Thumbs.db` — файлы с точным названием `Thumbs.db`;
- `*.tmp` — все файлы, названия которых заканчиваются на `.tmp`;
- `*.log` — все файлы, названия которых заканчиваются на `.log`.

## Правила файла terraform/.gitignore

- `**/.terraform/*` — игнорируются все файлы и каталоги внутри каталогов `.terraform` на любом уровне вложенности.
- `*.tfstate` — игнорируются все файлы, названия которых заканчиваются на `.tfstate`.
- `*.tfstate.*` — игнорируются все файлы, в названии которых после `.tfstate` находятся дополнительные символы или расширения.
- `crash.log` — игнорируются файлы с точным названием `crash.log`.
- `crash.*.log` — игнорируются файлы, названия которых начинаются с `crash.`, заканчиваются на `.log`, а между ними находятся любые символы.
- `*.tfvars` — игнорируются все файлы, названия которых заканчиваются на `.tfvars`.
- `*.tfvars.json` — игнорируются все файлы, названия которых заканчиваются на `.tfvars.json`.
- `override.tf` — игнорируются файлы с точным названием `override.tf`.
- `override.tf.json` — игнорируются файлы с точным названием `override.tf.json`.
- `*_override.tf` — игнорируются файлы, названия которых заканчиваются на `_override.tf`.
- `*_override.tf.json` — игнорируются файлы, названия которых заканчиваются на `_override.tf.json`.
- `.terraformrc` — игнорируются файлы с точным названием `.terraformrc`.
- `terraform.rc` — игнорируются файлы с точным названием `terraform.rc`.
- `*.tfplan` — игнорируются все файлы, названия которых заканчиваются на `.tfplan`.
- `.terraform.tfstate.lock.info` — игнорируются файлы с точным названием `.terraform.tfstate.lock.info`.

Символ `*` означает любое количество любых символов.

Символ `**` позволяет применять правило к каталогам на любом уровне вложенности.
